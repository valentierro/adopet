# Adopet — Nota de versão para Google Play (1.0.32)

Use o texto abaixo no campo **"O que há de novo"** ao enviar a atualização para o Google Play.

## Versão resumida (tudo incluído, texto enxuto)

```
• Sessão mais estável: não desloga em falhas temporárias de rede; ao expirar, redireciona para o feed sem alerta

• Menu inferior para visitante: "Entrar" no lugar de Perfil; abas Favoritos, Anunciar e Conversas levam ao login/cadastro
• Ao sair do detalhe do pet, visitante volta corretamente para o feed; ajustes de espaçamento no feed
```

## Versão longa (se o campo permitir mais texto)

```
Nesta atualização:

• Sessão e rede — A sessão fica mais estável e o app não desloga em falhas temporárias de rede. Quando a sessão expira, o app redireciona para o feed sem exibir mensagem de alerta.

• Versão do app — O número da versão passa a ser exibido nas telas de login, cadastro e boas-vindas (ex.: "Versão 1.0.32").

• Outras melhorias de estabilidade e experiência em diversos fluxos do app.
```
